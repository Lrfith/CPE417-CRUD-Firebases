# CPE417-CRUD-Firebases
This project was developed for HW12 in the CPE417 course at Sripatum University. It demonstrates the ability to build a full CRUD web application using Firebase Firestore and showcases the practical application of concepts learned throughout the course. Users can create, read, update, and delete records containing name and email information through a single-page interface.

## Features
- Create
-  Read 
- Update
- Delete
